## ⚡ Troubleshooting

If the software does not open:
1. Restart your computer.

3. Verify that you have enough space on your hard drive. 
4. Run the installer again if a file appears missing.

If you experience crashes:
1. Reduce the number of open windows on your screen.
2. Check your power plan settings. Set your power plan to high performance for better results. 
3. Ensure no other heavy programs are running in the background.
